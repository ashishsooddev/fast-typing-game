# ⚡ SpeedType Arena

A fast-paced typing game built with HTML, CSS, and JavaScript.

## 🎮 Features
- 90 randomized words
- 99-second timer
- Live score tracking
- Background music
- Restart without reloading
- OOP Score class implementation

## 🚀 Play Game
[Live Demo](https://ashishsooddev.github.io/fast-typing-game/)

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript (OOP)

## 📦 Setup
1. Clone the repo
2. Open index.html in browser
